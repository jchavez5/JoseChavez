Tecnologias

HTML5

HTML es un lenguaje de programación que se utiliza para el desarrollo de páginas de Internet.

CSS3

CSS3 es la última evolución del lenguaje de las Hojas de Estilo en Cascada.

JAVASCRIPT

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript.

JQUERY

jQuery es una biblioteca de JavaScript rápida, pequeña y rica en funciones. Hace cosas como el recorrido y manipulación de documentos HTML.

BOOTSTRAP

Cree proyectos receptivos y de primer nivel en dispositivos móviles en la web con la biblioteca de componentes front-end más popular del mundo.

ANDROID

Android es un sistema operativo inicialmente pensado para teléfonos móviles, al igual que iOS, Symbian y Blackberry OS. tecnologia nueva.

La web

La naturaleza abierta de la web ofrece increíbles oportunidades para los desarrolladores que desean crear sitios web o aplicaciones online.
